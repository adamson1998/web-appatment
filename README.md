# Goba Peaks Website

A premium residential website showcasing elevated living in Dar es Salaam, Tanzania.

## Overview

This website represents the Goba Peaks brand - a premium residential and apartment development created for individuals and families who appreciate contemporary design and elevated city living. The website follows the official Goba Peaks Brand Guidelines to ensure consistency and sophistication across all digital touchpoints.

## Features

### Design & Branding
- **Brand Compliant**: Follows official Goba Peaks brand guidelines
- **Color Palette**: Sunset Orange (#FF7E29), Sunset Pink (#FF4EBC), Deep Peak Blue (#0C2340)
- **Typography**: Montserrat font family with proper weight hierarchy
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices

### Sections
- **Hero Section**: Stunning gradient overlay with property showcase
- **About**: Overview of elevated living experience
- **Apartments**: Showcase of different unit types (Studio, 1BR, 2BR, 3BR, Penthouse)
- **Gallery**: Interactive photo gallery with category filtering
- **Location**: Strategic positioning in Dar es Salaam
- **Contact**: Comprehensive contact information and inquiry forms

### Interactive Features
- **Mobile Navigation**: Responsive hamburger menu
- **Gallery Filtering**: Dynamic category-based image filtering
- **Form Validation**: Client-side validation for inquiry forms
- **Smooth Scrolling**: Enhanced navigation experience
- **Animation Effects**: Scroll-based animations and parallax effects
- **Contact Forms**: Quick inquiry and detailed contact forms

## Technical Stack

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with CSS Grid and Flexbox
- **Vanilla JavaScript**: No external dependencies
- **Responsive Design**: Mobile-first approach
- **Performance Optimized**: Optimized images and efficient code

## Customization

### Colors
All brand colors are defined as CSS variables in `:root` for easy customization:
```css
:root {
  --sunset-orange: #FF7E29;
  --sunset-pink: #FF4EBC;
  --deep-peak-blue: #0C2340;
  --classic-white: #FFFFFF;
  --charcoal-gray: #2B2B2B;
}
```

- Update contact information in the contact section
- Modify apartment details and pricing
- Customize company information in the footer
