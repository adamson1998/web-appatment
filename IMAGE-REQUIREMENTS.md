# Goba Peaks Website - Image Asset Requirements

This document outlines all the required images for the Goba Peaks website to function properly.

## Image Specifications

### Format Guidelines
- **Logos**: SVG format (scalable, crisp at all sizes)
- **Photos**: JPG format (high quality, optimized for web)
- **Icons**: SVG format (scalable, brand-consistent)
- **Graphics**: PNG format (if transparency needed)

### Resolution Guidelines
- **Hero Images**: 1920x1080px minimum (Full HD)
- **Gallery Images**: 800x600px minimum
- **Icons**: 64x64px minimum (SVG scalable)
- **Logos**: Vector format (SVG) for infinite scalability

## Image Style Guidelines

### Photography Style
- **Lighting**: Natural light preferred, golden hour for exteriors
- **Composition**: Clean, uncluttered spaces
- **Color Grading**: Warm tones to complement brand palette
- **Perspective**: Wide angles to show space, architectural details
- **People**: If included, diverse, professional, aspirational

### Brand Consistency
- **Color Temperature**: Warm, inviting tones
- **Contrast**: High enough for premium feel
- **Saturation**: Natural but slightly enhanced
- **Shadows**: Soft, not harsh
- **Highlights**: Bright but not blown out

## File Naming Convention

Use consistent naming for easy maintenance:
- **Descriptive names**: `penthouse-living.jpg` not `IMG001.jpg`
- **Lowercase**: All lowercase letters
- **Hyphens**: Use hyphens for spaces
- **Consistent format**: `category-description-number.extension`

## Image Optimization

### Before Upload
- **Compress**: Optimize file sizes for web (80-90% quality for JPGs)
- **Resize**: Use appropriate dimensions for intended use
- **Format**: Choose correct format (JPG for photos, SVG for logos/icons)
- **Alt Text**: Prepare descriptive alt text for each image

### Quality Standards
- **JPG Quality**: 85-90% for main images, 80% for gallery thumbnails
- **PNG**: Use only when transparency is required
- **SVG**: Optimize code, remove unnecessary elements
- **Progressive JPG**: Use for large images to improve loading experience

## Responsive Considerations

### Multiple Sizes (Optional)
For optimal performance, consider providing multiple sizes:
- **Desktop**: Full resolution
- **Tablet**: 75% size
- **Mobile**: 50% size

### Critical Images
Priority loading order:
1. Logo and favicon
2. Hero image
3. Above-the-fold apartment images
4. Gallery and other images
